Basic Elements of HTML

Headers Paragraphs Hyperlinks Lists Images

Headers

The <header> element represents a container for inductory content or a set of navigational links. A header element typically contains:

  • One or more heading elements; h1 to h6
  • logo or icon
  • authorship information

You can have as many header elements as you want.

Paragraphs

The <p> tag defines a paragraph

Browsers automatically add some space (margin) before and after each paragraph element. The margins can be modified with CSS (with the margin properties.

Hyperlinks

The <a> tag defines a hyperlink, which is used to link from one page to another. By default links will appears as follows in browsers:

  • An unvisited link is underlined and blue
  • A visisted link is underlined and purple
  • An active link is underlined and red

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

Lists

There are two types of lists. Unordered lists use the <ul> element tag. Ordered lists use the <ol> element tag.


            <ol>
                <li>Start car</li>
                <li>Put car in drive</li>
                <li>Use gas pedal</li>
            <ol>
            
            <ul>
                <li>Apples</li>
                <li>Carrots</li>
                <li>Milk</li>
            <ul>
  1. Start car
  2. Put car in drive
  3. Use gas pedal
  • Apples
  • Carrots
  • Milk

Images

The <img> tag defines an image in an HTML page. It requires two attributes src and alt.

In HTML the <img> tag has no end tag.
